## Distribuição

Ao distribuir o binário do SaveSentry, os seguintes arquivos devem ser
incluídos no pacote:

- `SaveSentry.exe` — O executável principal
- `LICENSE` — Licença MIT do SaveSentry
- `THIRD-PARTY-LICENSES.md` — Atribuições e licenças das dependências

O script `build-windows.ps1` já inclui automaticamente esses arquivos no ZIP
de release.

## Nota

Este projeto é distribuído na esperança de que seja útil, mas SEM NENHUMA
GARANTIA. Consulte os arquivos de licença individuais de cada dependência
para os termos completos.

Para gerar uma lista atualizada de dependências e licenças, execute:
```bash
cargo tree --format "{p} {l}" --prefix none
```
